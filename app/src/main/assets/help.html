Use the top right menu to navigate between the voice and to-do activities.

Main screen

The last query executed will be displayed on a card, tap it to re-execute it.

Cards will be created for location and weather queries, long tap to remove them.

To-do list

Add items by using the add button.

Long tap an item to remove it.

Use the clear button to remove all items.

Voice commands

Activate voice commands by tapping the microphone icon in the main view.

Command - Help

For example: "Open help" or "Help"

Opens this screen.

Settings

Command - Show default location

Shows the default location for weather forecasts.

Command - Set default location to {location}

For example: "Set default location to Barcelona, Spain."

Sets the default location for weather queries, if this value is not specified, the weather service will use the device location to retrieve forecasts. Sometimes the service needs more information for a given city, in that case specifying the country will help.

If location permission is not granted, it will use Barcelona as default location for test purposes.

Command - Remove default location

Removes the default location for weather queries.

Command - Location

For example: "Show me my location"

Shows a card with current latitude, longitude, address and city.

It will show last known location if GPS not activated and current one if it is.

It requires location permissions to work.

Command - Calendar or Event

For example: "Open calendar" or "Add new event"

Opens calendar app to add a new event.

It requires calendar permissions to work

Command - To do

For example: "Open todo list", "To do"

Command - Weather

For example: "Show me the weather", "Weather" or "Current weather"

Displays the current weather

If default location is set, it will display the current weather for that location. If it's not, it will display current weather for last known location, or current location if GPS is activated.

If location permissions are not granted and default location is not set, it will display weather for Barcelona for demo purposes.

Command - Weather tomorrow

For example: "Show me the weather for tomorrow", "Weather tomorrow"

Displays the weather forecast for tomorrow

If default location is set, it will display the forecast for that location. If it's not, it will show forecast for last known location, or current location if GPS is activated.

If location permissions are not granted and default location is not set, it will display forecast for Barcelona for demo purposes.
